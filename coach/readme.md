## Contrôles manette
Si une manette est détectée:
* CARRE : stopper tous les robots
* CROIX : relance les robots
* R1 : but pour les jaunes
* L1 : but pour les bleus
* OPTIONS : reset match
* flèche du bas : arrêt du programme python
